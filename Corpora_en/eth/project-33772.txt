3D medical scanner and virtual patient framework for computer assisted assessment and evaluation of prosthetic and cranio-maxillofacial treatment and surgery In this project ETHZ, USZ and Disney Research will develop best in class 3D face scanning system tailored for medical applications. The scanning system consists of a set of hardware components and a software toolkit that will faciliate the scanning process for medical professionals. In this project ETHZ, University of Zurich and Disney Research will develop best in class 3D face scanning system tailored for medical applications and will commercialize it through an ETH spin-off company. The scanning system consists of a set of hardware components taylored to the requirement of medical face scanning and a software toolkit that will faciliate the scanning process for medical professionals.
