Zürcher Hochschule für Angewandte Wissenschaften: Projekte

Statistik frei praktizierender Hebammen in der Schweiz 2012

Statistics of independent midwives in Switzerland 2012

Projektleiter/in / Project leader: Claudia König

Projektteam / Project team: Marlen Amsler, Monika Schmid

Projektbeginn / Start date of project: 01.02.2013

ProjektpartnerInnen / Project partner(s): Schweizerischer Hebammenverband SHV

Seit 2005 wird vom Schweizerischen Hebammenverband die „Tätigkeitserfassung der frei praktizierenden Hebammen in der Schweiz„ durchgeführt. Die ZHAW hält seit 2009 das Mandat für die Aufbereitung und Auswertung der Daten sowie der Berichterstellung. Im 2012 wurden die Daten erstmals über das neu entwickelte Online-Tool erfasst.
