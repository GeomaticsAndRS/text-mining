expert-match

Welcher Kandidat bietet den besten Match? Intelligentes System zur Effizienzsteigerung in der Suche von Fachspezialisten

Which candidate is the best match? Intelligent system for efficiency improvement of expert search

Projektleiter/in / Project leader: Martin Braschler

Projektteam / Project team: Alexander Horvath, Thierry Musy, Stefan Rietberger

Projektfinanzierung / Financing: KTI

Projektbeginn / Start date of project: 01.01.2012

ProjektpartnerInnen / Project partner(s): expert group ag

Die expert group ag hat in ihrem Geschäftsbereich des „Expert und Executive Search“ ein Defizit an guten Suchlösungen identifiziert. Deshalb wird zur Qualifikation von Kandidatenprofilen zwecks Rekrutierung von hochspezialisierten Fachkräften das System „expert-match“ erarbeitet. Mit Hilfe von State-of-the-Art Information Retrieval-Methoden wird der Suchprozess der expert group ag verbessert. Der Prozess wird effektiver und effizienter, so dass sowohl Kosteneinsparungen wie auch eine weitere Steigerung der Dienstleistungsqualität erreicht werden können. Die Innovation besteht in der Anwendung von spezifisch abgestimmten IR Methoden auf den vorliegenden Geschäftsfall, sowie der Parametrisierung und Evaluation des Systems.
