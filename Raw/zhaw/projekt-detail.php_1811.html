Stadtplan Wintertur

Stadtplan Winterthur

Projektleiter/in: Thomas Müller

Projektbeginn: 23.01.2002

ProjektpartnerInnen: Informatikdienste der Stadt Winterthur (IDW)

Weiterführende Informationen: http://citymap.zhwin.ch/servlet/Citymap

Die ZHW betreibt einen Online-Stadtplan der Stadt Winterthur. Dieser ist im Internetauftritt der Stadt Winterthur eingebunden.

Im Rahmen einer Diplomarbeit 2000 wurden die existierenden Versionen überarbeitet und an die neuen Bedürfnisse angepasst. Insbesondere wurde das Datenbank-Schema neu erstellt und der Code nach JAVA portiert. Neu wurde für die geometrischen Daten die Oracle Spatial Data Option verwendet. Des weiteren können für jedes Objekt beliebig viele Informationen eingegeben werden. Geografische Daten einer Stadt ändern ständig. Ein Tool zum Aktualisieren dieser Daten drängte sich also auf. Die Diplomanden legten den Grundstein zum Administrations Tool (von Ihnen Management Tool genannt).

Die aktuelle Arbeit befasst sich mit der Installation in der neuen Betriebsumgebung der ZHW und der Inbetriebnahme sowie der nötigen Überarbeitung der Diplomarbeit. Im Rahmen des neuen Internetauftrittes der Stadt werden weitere Anpassungen nötig.

Das Datenbank-Schema und einzelne JAVA-Klassen (z.B. Ressourcenmanager) der Diplomarbeit wiesen schwerwiegende Mängel auf, so dass sie im Rahmen dieser Arbeit überarbeitet werden müssen.

Weiter kann auf die aktuellen Bedürfnisse des Auftraggebers (Informatikdienste der Stadt Winterthur, IDW) eingegangen werden.
