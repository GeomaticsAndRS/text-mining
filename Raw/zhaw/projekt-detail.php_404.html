Zürcher Hochschule für Angewandte Wissenschaften: Projekte

Entwicklung eines auf Bayes Netzwerken basierenden Expertensystems und seiner Anwendung in einer konkreten Marktsituation.

Design of an online expert system based on Bayes’ian Networks and its first market implementation

Design of an online expert system based on Bayes’ian Networks and its first market implementation

Projektleiter/in / Project leader: Marcel Burkhard, Thomas Eggel

Projektteam / Project team: Roland Gassmann, Thomas Ott, Erich Zbinden

Projektfinanzierung / Financing: KTI

Projektvolumen / Project budget in SFr.: 1 052 000

Projektbeginn / Start date of project: 01.01.2013

ProjektpartnerInnen / Project partner(s): dayzzi (Schweiz) AG / IfM-HSG

Im Rahmen eines Knowledge Engineering Projektes wird im Gebiet eines stark segmentierten B2B Marktes ein online Expertensystem entwickelt, das den aufwändigen Verkaufsprozess effizienter und präziser gestaltet, indem sowohl die Beratungsqualität erhöht als auch die Beratungskosten gesenkt werden.
Im Zentrum der Plattform steht ein Expertenalgorithmus, der aufbauend auf Machine Learning Techniken die Komplexität des Beratunsprozesses bewältigen und mithilfe von neuen bio inspired Algorithmen eine wirksame Prozessabwicklung garantieren kann.
